# Tokens

js-ds-ui uses DTCG-compliant tokens for color, spacing, sizing, and typography.
Use semantic tokens whenever possible to stay aligned with themes.

## Examples

```css
.card {
  background: var(--color-background-secondary);
  padding: var(--space-component-padding-lg);
  border-radius: var(--component-button-border-radius);
}
```
